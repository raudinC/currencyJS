# currencyJS

currencyJS is a JavaScript library for currency formatting.

## Version

0.1

## Installation

In a browser:

```
<script src="currencyJS.js"></script>
```

In an AMD loader:

```
require(['currencyJS'], function(currencyJS) {});
```


## Methods

### Format

```
currencyJS.format(2252.25, "dollar"); //$2,252.25
```

Valid currencies: 

- dollar

- euro
